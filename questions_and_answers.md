## 1. How many hours has Alice Johnson worked this week?

**Status:** Succeeded

**SQL:**
```sql
SELECT SUM((JULIANDAY(WorkSession.endTime) - JULIANDAY(WorkSession.startTime)) * 24) AS hours_worked
FROM WorkSession
JOIN Person ON WorkSession.personId = Person.id
WHERE Person.firstName = 'Alice' AND Person.lastName = 'Johnson'
AND WorkSession.startTime >= DATETIME('now', 'start of day', '-6 days');
```

**Raw Result:** `[(6.50000000372529,)]`  
**Friendly Response:** Alice Johnson worked approximately 6.5 hours this week.

## 2. Which projects involve security?

**Status:** Succeeded

```sql

JOIN ProjectTag ON Project.id = ProjectTag.projectId
```

## 3. What is the total cost of the Data Migration project?

**Status:** Succeeded

**SQL:**
```sql
SELECT SUM((JulianDay(endTime) - JulianDay(startTime)) * 24 * hourlyRate) AS total_cost
FROM WorkSession
JOIN PersonProject ON WorkSession.personId = PersonProject.personId AND WorkSession.projectId = PersonProject.projectId
JOIN Project ON WorkSession.projectId = Project.id
WHERE Project.projectTitle = 'Data Migration';
```

**Raw Result:** `[(347.49999962747097,)]`  
**Friendly Response:** The total cost of the Data Migration project is approximately $347.50.

## 4. Who has contributed to more than one project?

**Status:** Succeeded

**SQL:**
```sql
SELECT Person.firstName, Person.lastName, Person.email
FROM Person
JOIN PersonProject ON Person.id = PersonProject.personId
WHERE PersonProject.role = 'Contributor'
GROUP BY Person.id
HAVING COUNT(DISTINCT PersonProject.projectId) > 1;
```

**Raw Result:**  
`[('Alice', 'Johnson', 'alice@pmtool.com'), ('Bob', 'Lee', 'bob@pmtool.com'), ('Clara', 'Nguyen', 'clara@pmtool.com'), ('Derek', 'Stone', 'derek@pmtool.com'), ('Emily', 'Clark', 'emily@pmtool.com')]`  
**Friendly Response:** Alice, Bob, Clara, Derek, and Emily have all contributed to more than one project.
